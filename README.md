* Instalación de Nodjes

* Instalación de CLI SAM
  * https://docs.aws.amazon.com/serverless-application-model/latest/developerguide/serverless-sam-cli-install.html

## Instalación

ejecutar comando

```
npm install
```

## Levantar proyecto

```
npm run develop
```
